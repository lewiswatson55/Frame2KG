# Frame2KG Documentation Hub

Frame2KG aggregates the data, models, and tooling introduced in *Frame2KG-YC2: A Synthetic Dataset, LoRA Adapters, and an Evaluation Toolkit for Frame → Graph*. The goal is to provide a single launch point for reproducible frame-to-knowledge-graph research focused on interpretable, on-device robotics.

## Project Resources
- Dataset: Frame2KG-YC2 synthetic corpus ([hidden for anonymous review](https://hf.co))
- Training pipeline: Frame2KG-Trainer for Qwen2.5-VL LoRA adapters ([coming soon](https://example.com/))
- Evaluation toolkit: Deterministic frame2kg-eval matcher and metrics ([coming soon](https://example.com/eval))
- Adapters: Released LoRA checkpoints (3B/7B; QKVO ± GateProj) ([hidden for anonymous review]((https://hf.co)))
- Benchmarks: Frame2KG baseline reports and comparison tables ([coming soon](https://example.com/))

> Note: Resources marked "hidden for anonymous review" are hosted on huggingface and cannot be anonymously accessed.
